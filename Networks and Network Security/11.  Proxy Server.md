# Proxy Server
A proxy server is a server that acts as a middleman between your device and the internet. Instead of connecting directly to a website, your request first goes to the proxy server, 
and then the proxy server connects to the website on your behalf. For Example: Imagine you want to send a letter to someone but don't want them to know your address. So you give 
the letter to a middleman who sends it from their own address. The recipient only sees the middleman's address — not yours. That middleman is the proxy server.

## How a Proxy Server Works
A proxy server sends and receives requests on behalf of your network. As a result, external systems cannot see the real IP Address of your internal servers. This creates an additional 
security layer through IP masking.

### The Request-Response Cycle
- **_Interception:_** When you request a resource (e.g., a specific web page), your browser is configured to send that request to the Proxy IP instead of the destination IP.
- **_Identity Substitution:_** The proxy receives your packet, strips away your original IP Address, and replaces it with its own. This is a form of Network Address Translation (NAT) at the application or transport level.
- **_Encapsulation & Forwarding:_**The proxy forwards the modified request to the web server. To the destination, it appears that the proxy server is the one requesting the data.
- **_Validation & Delivery:_** The web server sends the response back to the proxy. The proxy inspects the returning data for malicious signatures (if it has security features enabled) and then routes it back to your internal device.

## Technical benefits
- **_Caching (Performance):_** Proxies store "local copies" of frequently accessed websites. If 50 employees visit the same news site, the proxy only downloads the data once from the internet and serves it 49 times from its local disk, saving massive bandwidth.
- **_Anonymity (Privacy):_** By acting as a shield, it prevents external servers from knowing your internal network topology or specific device details.
- **_Access Control (Policy):_** Administrators use proxies to enforce "Blacklists" (blocking harmful sites) or "Whitelists" (allowing only specific work-related domains).

## Types of Proxy server
**_In real-world environments, there are more than a dozen proxy variations, and they are often used together in hybrid architectures. A proxy is not a complete security solution—it 
is a visibility and control point. If misconfigured, it can become an attack surface itself._**

## Functional Types of Proxies
### Forward Proxy: 
Used to control traffic from internal users to the internet.
- Function: Hides user IP addresses and manages outgoing requests
- Security: Blocks malicious or unsafe websites using URL filtering
  
### Reverse Proxy
Handles incoming traffic from the internet before it reaches internal servers.
- Function: Accepts and validates incoming requests, then forwards them to backend servers
- Security: Protects internal server identities and prevents direct exposure

### Transparent Proxy
Operates without user awareness
- Common in ISPs and public Wi-Fi networks

### Anonymous Proxy
- Hides the user’s IP but reveals that a proxy is being used

### Elite / High-Anonymity Proxy
Hides both the user’s IP and the fact that a proxy is in use
- Makes tracking significantly harder

## Protocol-Based Proxies
###  HTTP Proxy
- Handles only HTTP web traffic

###  HTTPS Proxy
- Handles encrypted web traffic (TLS/SSL)

###  SOCKS Proxy (SOCKS4 / SOCKS5)
Highly flexible
- Supports TCP and UDP traffic
- Often used in advanced use cases (e.g., tunneling, evasion)

## Deployment / Infrastructure-Based Proxies
###  Data Center Proxy
Hosted in cloud/data centers
- High speed but easier to detect

###  Residential Proxy
Uses real ISP-assigned IP addresses
- Harder to detect, often abused for scraping and evasion

###  Mobile Proxy
Uses mobile carrier IPs (4G/5G)
- Highly trusted in many systems

###  Public Proxy
Free and open for anyone
- High risk (logging, malware injection, lack of control)

###  Private / Dedicated Proxy
Assigned to a single user
- More secure and reliable

## Specialized Security Proxies
###  Web Proxy
- Used for filtering and monitoring web activity

###  Email Proxy
- Filters spam and detects phishing attempts

###  DNS Proxy
- Filters domain requests at the DNS layer

###  SSL Inspection Proxy
Decrypts and inspects encrypted traffic
- Critical in enterprise SOC environments for threat detection
